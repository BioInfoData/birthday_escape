# חדר בריחה - יום הולדת (Hebrew Birthday Escape Room)

אתר חידות ליום הולדת, מותאם לילדים, עובד ללא שרת (Static Site) וקל להתאמה אישית.

## 🚀 איך מפעילים? (הכי מהר שאפשר)

1. **הורדה:** הורידו את כל הקבצים למחשב שלכם.
2. **הפעלה מקומית:** פתחו את הקובץ `index.html` בדפדפן (Chrome/Edge/Safari).
   - *שימו לב:* במחשב המקומי (ללא שרת) ייתכן וחלק מהמעברים לא יעבדו חלק בגלל הגבלות אבטחה של הדפדפן (בעיקר סביב localStorage או מעבר בין קבצים). ההמלצה היא תמיד להריץ על שרת מקומי או להעלות ל-GitHub Pages.

## 🌐 איך מעלים לאינטרנט בחינם (GitHub Pages)?

זה הפתרון הכי טוב ויציב.
1. צרו חשבון ב-[GitHub](https://github.com).
2. צרו **Repository** (מאגר) חדש (Public), תנו לו שם (למשל `birthday-game`).
3. העלו את כל הקבצים שיצרנו לכאן (או דרך הממשק בדפדפן - Upload Files, או דרך Git).
   - ודאו שהקובץ `index.html` נמצא בתיקייה הראשית (Root).
4. לכו ללשונית **Settings** -> בתפריט צד בחרו **Pages**.
5. תחת **Branch**, בחרו ב-`main` (או `master`) ולחצו **Save**.
6. תוך דקה תקבלו קישור לאתר (למשל `https://username.github.io/birthday-game/`). שלחו אותו למשתתפים!

## 🛠️ איך משנים את החידות?

כל התוכן נמצא בקובץ אחד שקל לערוך: `assets/game-config.js`.
פתחו אותו עם כל עורך טקסט (Notepad, VS Code).
שם תוכלו לשנות:
- את הכותרות והסיפורים.
- את הקודים (התשובות הנכונות).
- את הרמזים.
- את הפרס בסוף.

## 🎨 תמונות

- שמו את התמונות שלכם בתיקייה `assets/img`.
- הקפידו ששמות הקבצים יהיו באנגלית (למשל `stage1.jpg`).
- עדכנו את `game-config.js` כדי שיצביע לתמונות הנכונות.

## 🆘 גיבוי

אם האינטרנט נופל באמצע המסיבה, הדפיסו מראש את הקובץ `backup.html`. הוא מכיל את כל החידות, הרמזים והפתרונות בצורה נוחה.
